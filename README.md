## AI Movie Recommendation App

An advanced React-based movie app that allows users to:

* Search movies using TMDB API
* Get AI-powered movie suggestions based on mood
* Save favorite movies
* Experience smooth infinite scrolling


## Features

# Movie Search

* Search movies by title
* Debounced input for better performance

###  AI Mood Matcher

* Enter your mood (e.g., *"sad but want action movie"*)
* AI suggests a movie title
* Automatically fetches movie details from TMDB


###  Infinite Scrolling

* Loads more movies automatically while scrolling

### Favorites System

* Save/remove movies from favorites using localStorage

###  Performance Optimizations

* Lazy loading images (`loading="lazy"`)
* Debouncing API calls
